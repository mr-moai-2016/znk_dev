はじめに
    Moaiはポータブルウェブサーバシステムです.

    最初にmoaiというプログラムを一つ起動します.
    次に Moaiトップページ へアクセスしましょう！
    (このページと同様の画面が表示されたならmoaiは正常に起動しています)

    moaiが起動中は、あなたのマシンはローカルなプロキシサーバとウェブサーバとして同時に機能します(※1).
    これにより様々なWebアプリケーションをあなたのマシンだけで完結した形で、ブラウザ上にて利用できるようになります.

    以下でご紹介するツール群は、このWebアプリケーションの応用例です.
    それではさっそくMoai、そして今回のバージョンの目玉機能であるEasterの世界をご紹介しましょう！

    ※1) デフォルトの設定では安全のため、あなたのマシン以外からのMoaiへの接続はすべて遮断されるようになっています.

Webアプリケーション
  • Easter (Author: Zenkaku)
    • 概要

    Easterは仮想ブラウザ環境(以降、単に仮想環境と呼ぶ)を利用するツールです.

    Moai Ver1.1.x系まではこの仮想環境を利用するには、ブラウザにて「ローカルプロキシの設定」を行っておく必要がありました(※1).
    Ver2.0以降で導入されたEasterを使えば、この設定を行うことなく即座に仮想環境を利用することができます(※2).
    さっそく上のリンクよりEasterへ入り仮想環境を体験してみましょう…とその前に、下記のVirtualUSERSも一読しておきましょう！

    ※1) もちろん現バージョンでも旧来通りMoaiをローカルプロキシとして使うこともできます.
    　　Easterの場合は使用可能なサイトが限られますが、一方でローカルプロキシの場合は基本的にどのようなサイトにでも使用可能です.
    ※2) Moai Ver2.0で新たに導入されたXhrDMZシステムは、XMLHttpRequestによる攻撃を防御します.
    　　これによりEasterによるウェブ閲覧の安全性を高めています.

  • VirtualUSERS (Author: K.Yakisoba.H and Zenkaku)
    • VirtualUSERSは、ランダムな仮想環境を自動的に構築するツールです.
    これはボタン一つ押すだけで簡単に行えます.

    このボタンは別にいつ押しても構わないのですが、もっとも適切なタイミングはルータリセットなどによりIPアドレスを変更した直後です.
    また、Moaiをダウンロード後に初めて起動したなら、Moaiが提供する仮想環境は初期設定値のものになっているはずですから、
    この場合もまずはこのボタンを一回押して仮想環境の値を変更しておきましょう！

  • CustomBoy (Author: K.Yakisoba.H and Zenkaku)
    • CustomBoyとは、Moai/Easterの仮想環境を詳細にカスタマイズするエキスパートユーザ向けのツールです.
    (従って初心者の方がこのツールを使う必要はまずないと思われます)
    我々が提供する仮想化ツールの中でもこれは最強のカスタマイズ性を有するものです.

  • ProxyFinder (Author: Mr.Moai)
    • ProxyFinderは定番のプロキシ情報提供サイトへ接続し、現在有効なプロキシサーバのリストを自動的に取得するツールです.
    また、Moaiが使用するProxyサーバをリストから選択して設定することができます.

  • Moai CGI Developers (Author: Mr.Moai)
    • Moai CGI は Moaiとは独立した単独のCGIスクリプトを実行します.
    従って、あなたがCGIスクリプトを独自に開発し、それをMoai CGI上で実行させることもできるのです！
    ここではこのようなWebアプリケーションを独自に作ってみたい野心溢れる方向けに、その開発方法を解説します.
