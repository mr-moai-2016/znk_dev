Moai 次期バージョンの開発状況について

現在、Moaiの最新バージョンは2.1です.
次期バージョンMoai Ver2.2( コードネーム ResponsiveStone )のリリースに向け開発中です. 次回のバージョンでは以下の修正や新機能の導入を予定しています.


Androidに関する既存の問題点の大幅な修正
現状のMoai Ver2.1ではAndroidの新しいバージョン(5.0以降)におけるMoai起動の不具合などが確認されています. この不具合を全面修正します. また現状のMoai Ver2.1をAndroidで使用する場合ターミナルエミュレータを使う必要がありますが、 Moai Ver2.2ではそれを不要とし、通常のアプリ同様apkファイルをタップすることでインストールを可能とし、 アイコンをタップすることでMoaiを起動可能とする予定です.
ただしapkファイルのGoogle Play経由での配布はせず、Moai公式サイトからこれをダウンロードする形とします.
またレスポンシブデザインの導入により、スマホでのユーザーマニュアルやツールの画面の見やすさ、使いやすさを改良します.

スクリプト言語Raraku
Moai独自のスクリプト言語Rarakuを搭載し、それを用いてMoai CGIの開発などが行えるようになります.
