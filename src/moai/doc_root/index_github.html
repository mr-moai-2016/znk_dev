はじめに

我々「全日本ネットワーク研究所(通称ZNK)」では、フリーなアプリケーションを開発して皆様に提供しております.
これらはすべてオープンソースであり、またほぼすべてが Windows、Linux、そしてAndroid向けに開発されたものであります.
現在このサイトで提供しておりますものを以下でご紹介しましょう.

Moai Ver2.0のご紹介

  • Moai Ver2.0本体 (Author: Mr.Moai)
    概要

    これはまた一つ生まれたばかりのポータブルウェブサーバエンジンです.


    動作環境

    以下の環境で動作します.


    Windows2000以降

    Linux x86/x86_64

    Android 4.0以降


    スクリーンショット

    Moaiが起動中はあなたのブラウザでこのような画面を表示させることができ、
    ここからEasterやVirtualUSERSなどのWebアプリケーションをブラウザ上で利用できるようになります.
    ほとんどの方はこのEasterをメインで常時利用し、必要に応じてVirtualUSERSを使うという形になると思われます.
    (旧バージョンに引き続き、ローカルプロキシとして利用することもできます).




    インストール方法

    基本的にはzipファイルをダウンロードして展開し、moai(実行ファイル)を実行するだけです.
    こちらを参照してください.


    ドキュメント

    Moaiエンジン本体についてのリファレンスはこちらになります.

    Easterのご紹介

  • Easter (Author: Zenkaku)
    概要

    EasterはMoai Ver2.0上で動くウェブアプリケーションの一つです.
    あなたのブラウザ上で仮想ブラウザ環境を提供します.


    動作環境

    以下の環境で動作します.


    Moai CGI


    サポートサイト

    以下のサイトをサポートします.


    ふたばちゃんねる

    ５ちゃんねる


    スクリーンショット

    Easterの画面に入るとあなたのブラウザでこのような画面が表示され、
    ここから仮想ブラウザ環境にてリンク先のサイトを閲覧することができます.




    インストール方法

    Moaiエンジン本体をインストールすればデフォルトで付属しています.
    moaiを実行してブラウザからhttp://127.0.0.1:8124を開けば、あたかもmoaiの一機能のように動作します.


    ドキュメント

    Easterについてのリファレンスはこちらになります.

    VirtualUSERSのご紹介

  • VirtualUSERS (Author: K.Yakisoba.H and Zenkaku)
    概要

    VirtualUSERSはMoai Ver2.0上で動くウェブアプリケーションの一つです.
    Moai/Easterでの仮想ブラウザ情報をボタン一つで自動的に生成、再生成します.


    動作環境

    以下の環境で動作します.


    Moai CGI


    サポートサイト

    以下のサイトをサポートします.


    ふたばちゃんねる

    ５ちゃんねる


    スクリーンショット

    VirtualUSERSの画面に入るとあなたのブラウザでこのような画面が表示され、
    この画面の「VirtualUSERS」ボタンを押すだけで仮想ブラウザ環境の構築が完了します.
    設定等はほとんどなく、初心者でも簡単に使うことができます.




    インストール方法

    Moaiエンジン本体をインストールすればデフォルトで付属しています.
    moaiを実行してブラウザからhttp://127.0.0.1:8124を開けば、あたかもmoaiの一機能のように動作します.


    ドキュメント

    VirtualUSERSについてのリファレンスはこちらになります.

    Moaiで動作するその他のWebアプリケーション

  • CustomBoy (Author: K.Yakisoba.H and Zenkaku)
    概要

    Moai上で動くウェブアプリケーションの一つです.
    Moai/Easterの仮想ブラウザ情報を詳細にカスタマイズするVirtualUSERSより上級向けのツールです.
    我々が提供する仮想化ツールの中でもこれは最強のカスタマイズ性を有するものです.
    通常はVirtualUSERSで十分であり、初心者の方がこのツールを使う必要はまずないと思われます.


    インストール方法

    Moaiエンジン本体をインストールすればデフォルトで付属しています.
    moaiを実行してブラウザからhttp://127.0.0.1:8124を開けば、あたかもmoaiの一機能のように動作します.


    ドキュメント

    CustomBoyについてのリファレンスはこちらになります.
    (ただし内容はエキスパート向けです)


  • ProxyFinder (Author: Mr.Moai)
    概要

    Moai上で動くウェブアプリケーションの一つです.
    定番のプロキシ情報提供サイトへ接続し、現在有効なプロキシサーバのリストを自動的に取得します.
    Moaiで使用するProxyサーバの候補を作成/更新できます.


    インストール方法

    Moaiエンジン本体をインストールすればデフォルトで付属しています.
    moaiを実行してブラウザからhttp://127.0.0.1:8124を開けば、あたかもmoaiの一機能のように動作します.


    ドキュメント

    用意しておりません.
    しかしボタン一つしかないので迷うことはないと思います…


  • Moai CGI Developers (Author: Mr.Moai)
    概要

    Moai上で動くウェブアプリケーションは、あなたが独自に開発することもできるのです！
    これは単なるCGI(MoaiではMoai CGIと呼びます)であり、CGIの知識がある方なら特に難しいものではありません.
    Moai CGI DevelopersではこのようなWebアプリケーションを独自に作ってみたい方向けに、その開発方法を解説します.


    内容

    Moaiエンジン本体をインストールすればデフォルトでサンプルCGIとサンプルコード等が付属しています.
    これらのサンプルCGIはmoaiを実行してブラウザからhttp://127.0.0.1:8124を開けば、
    あたかもmoaiの一機能のように動作します.
    解説記事はこちらからご覧頂けます.

    • Moai 旧バージョン(非推奨)

  • Moai Ver1.1.8 (Author: Mr.Moai)
    概要

    このツールは一言で言えばHTTPを仲介するローカルプロキシです.
    同種のツールとしてはProxomitronやPolipoなどがあり、これらをご存知の方は
    それと基本的に同じようなものです.

    この旧バージョンは、まもなくサポートの終了を予定しています.
    現在旧バージョンをご利用の方はMoai Ver2.0への移行を強くお勧めします.
    （互換性はまったくなく、完全に別のツールと思っていただいた方がよいです)
    ちなみにMoai Ver2.0でもMoai Ver1.1.8と同様の使い方、即ちローカルプロキシとして使用することも可能です.


    ドキュメント

    Moai Ver1.1.8についての詳細はこちらになります.
    これは旧バージョン時代の古いドキュメントを若干手直ししたものです.
    Moai Ver2.0を使われる方にとっては参照する必要のないものですが、
    旧バージョンをこのまま使われ続ける方向けに一応このまま残します.
