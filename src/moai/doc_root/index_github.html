はじめに
    我々「全日本ネットワーク研究所(通称ZNK)」では、フリーかつオープンソースなアプリケーション
    およびライブラリを開発し、皆様に提供しております.
    これらはほぼすべてC/C++によってWindows またはLinux またはAndroid向けに開発されたものであります.
    現在このサイトで提供しておりますものを以下でご紹介しましょう.

Moai Ver2.0 本体 および 主要なWebアプリケーション
    Moai Ver2.0のインストールについてはインストールガイドをご覧下さい.
    Moai Ver2.0では以下で挙げたウェブアプリーションはデフォルトで付属しています.

  • Moai Ver2.0本体 (Author: Mr.Moai)
    • また一つ生まれたばかりのポータブルウェブサーバエンジンです.
    Moai Ver2.0が起動中は、様々なWebアプリケーションをあなたのブラウザ上にて利用できるようになります.
    以下でご紹介するツール群は、このWebアプリケーションの例です.

  • Easter (Author: Zenkaku)
    • Moai Ver2.0上で動くウェブアプリケーションの一つです.
    あなたのブラウザ上で仮想ブラウザ環境を提供します.

  • VirtualUSERS (Author: K.Yakisoba.H and Zenkaku)
    • Moai Ver2.0上で動くウェブアプリケーションの一つです.
    Moai/Easterでの仮想ブラウザ情報をボタン一つで自動的に生成、再生成します.

  • CustomBoy (Author: K.Yakisoba.H and Zenkaku)
    • Moai上で動くウェブアプリケーションの一つです.
    Moai/Easterの仮想ブラウザ情報を詳細にカスタマイズするVirtualUSERSより上級向けのツールです.
    我々が提供する仮想化ツールの中でもこれは最強のカスタマイズ性を有するものです.
    通常はVirtualUSERSで十分であり、初心者の方がこのツールを使う必要はまずないと思われます.

Moai Ver2.0 その他のWebアプリケーション
    Moai Ver2.0では以下で挙げたウェブアプリーションはデフォルトで付属しています.

  • ProxyFinder (Author: Mr.Moai)
    • Moai上で動くウェブアプリケーションの一つです.
    定番のプロキシ情報提供サイトへ接続し、現在有効なプロキシサーバのリストを自動的に取得します.
    Moaiで使用するProxyサーバの候補を作成/更新できます.

  • Moai CGI Developers (Author: Mr.Moai)
    • Moai上で動くウェブアプリケーションは、あなたが独自に開発することもできるのです！
    これは単なるCGI(MoaiではMoai CGIと呼びます)であり、CGIの知識がある方なら特に難しいものではありません.
    Moai CGI DevelopersではこのようなWebアプリケーションを独自に作ってみたい方向けに、その開発方法を解説します.

C/C++用ライブラリ、プラグイン
  • libZnk (Author: Zenkaku)
    • C/C++でプログラミングする上で基本的なものを集めたライブラリです.
    このリポジトリにあるほぼすべてのプログラムはこれを使って開発されています.

  • libRano (Author: Zenkaku and Mr.Moai)
    • ウェブサーバやウェブアプリケーションの開発をサポートするC/C++用ライブラリです.
    MoaiやEaster、CustomBoyなどのMoai CGIはこれを使って開発されています.

  • Moai plugins (Author: Zenkaku and Mr.Moai)
    • Moai用のプラグインで、Moaiを匿名掲示板で使うためのものとなります.
    現在提供しているのは futaba(ふたばちゃんねる用)と5ch(５ちゃんねる用)の二つです.
    Moai Ver2.0ではこれらはデフォルトで付属しています.

Moai 旧バージョン(非推奨)
  • Moai Ver1.1.8 (Author: Mr.Moai)
    • このツールは一言で言えばHTTPを仲介するローカルプロキシです.
    同種のツールとしてはProxomitronやPolipoなどがあり、これらをご存知の方は
    それと基本的に同じようなものです.

    この旧バージョンは、まもなくサポートの終了を予定しています.
    現在旧バージョンをご利用の方はMoai Ver2.0への移行を強くお勧めします.
    （互換性はまったくなく、完全に別のツールと思っていただいた方がよいです)
    ちなみにMoai Ver2.0でもMoai Ver1.1.8と同様の使い方、即ちローカルプロキシとして使用することも可能です.
