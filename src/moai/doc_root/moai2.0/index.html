ユーザーマニュアルのご案内

このユーザーマニュアルでは、ほとんどの方にとって有用なMoaiの機能に限定して、
チュートリアル的な解説記事を掲載しています.

上のリンクは階層構造となっており、現在あなたは「ユーザーマニュアル」の「入り口」を閲覧されている状態です.
「ユーザーマニュアル」には多くの記事があり、上のリンクの２段目に列挙されているものたちがそれに該当します.
ここではそれらの概要を紹介しましょう.
  • インストールガイド
    まずはMoaiをインストールしなければ何も始まりませんね！
    ここでは最新版のMoaiをインストールしてそれを初めて起動するまでを解説してあります.

    インストールといっても、Moai公式サイトより配布されている実行バイナリのzipファイルをダウンロードし、
    それを展開してmoaiという実行ファイルを実行するだけの単純なものです.

    Moaiはマルチプラットフォームであるため、それらはWindows, Linux, Androidなど環境によって異なります.
    よってそれぞれの環境について別途解説してあります.

  • Moaiの特徴
    他ツールと比べた場合におけるMoaiの特徴について端的にまとめてあります.

  • Easter
    Easterは仮想環境を利用してサイトを閲覧するためのビューアです.
    これを仮想ブラウジングと呼びます.
    ここではEasterの仮想ブラウジングやその他の機能について解説しています.

    ほとんどの方はこの仮想ブラウジング機能をメインで使う形になると思います.
    他にも画像の分類や管理を提供するコレクション機能がありますが、
    これを使わない方は気にする必要はありません.

  • VirtualUSERS
    VirtualUSERSは仮想環境をボタン一発で簡単構築するためのツールです.
    ここではこのツールの使い方を解説しています.

    ほとんどの方はこのVirtualUSERSをMoaiをインストール直後最初に一回実行し、
    以降は必要に応じて、主にルータリセット等でIPアドレスを変更した直後に実行する形になると思います.

  • Moaiのアップグレード方法
    Moai Ver2.1より導入された「Moaiアップグレード」はボタンを押すだけでMoaiシステム全体を簡単アップグレードするための機能です.
    ここではこれについて解説しています.
    また、Moai Ver2.0から Moai Ver2.1へアップグレードする方法についても説明してあります.
